Mephisto¶

About¶

Mephisto is a Mozilla Firefox 4 Add-On that adds an HTTP server inside your browser.

This server is able to open a webpage on the browser and returns content information or a screenshot.

Features¶

  • Return DOM as a human user will get it;
  • Return content information as JSON;
  • Modify DOM and/or results with external scripts;
  • Take screenshots.
